# java-sv2-adv-project-SzalayTamas
Vizsgaremek,elektromos művek ügyfélnyilvántartó alkalmazás
A program képes lesz tárolni az ügyfeleket, elérhetőségüket,hogy az ügyfél mennyi áramot fogyasztott.Képes a fogyasztási adatok alapján számlát készíteni és nyomon lehet követni mely számlák vannak már befizetve. Ügyfél minden adata modosítható , egyszerüen lehet rőgízeni ügyfeleket,vilanyóra leolvasásokat.
